# AlternateHistories

A console-based application for exploring alternate history scenarios through interactive storytelling powered by AI.

## Features

- **Interactive Historical Scenarios**: Choose from pivotal moments in history
- **AI-Powered Storytelling**: Dynamic choices and consequences generated by OpenAI GPT-4
- **Multiple Timelines**: Explore how different decisions reshape history
- **Console Interface**: Clean, colorful command-line experience

## Available Scenarios

1. **The Library of Alexandria** - Prevent the destruction of ancient knowledge
2. **Mongol Invasion of Europe** - Command the Mongol expansion into Europe
3. **Columbus Sails West** - Navigate the unexpected discovery of new lands
4. **The Black Death Prevention** - Use medieval medicine to stop a pandemic

## Setup

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd AlternateHistories
   ```

2. **Install dependencies**
   ```bash
   pip install -r requirements.txt
   ```

3. **Set up environment variables**
   ```bash
   copy .env.example .env
   ```
   Edit `.env` and add your OpenAI API key:
   ```
   OPENAI_API_KEY=your_openai_api_key_here
   ```

4. **Run the application**
   ```bash
   python main.py
   ```

## Requirements

- Python 3.7+
- OpenAI API key
- Internet connection for AI interactions

## How It Works

1. **Choose a Scenario**: Select from historical turning points
2. **Make Decisions**: At key moments, choose from AI-generated options
3. **See Consequences**: Watch how your choices ripple through history
4. **Explore Alternatives**: Try different paths to see various outcomes
